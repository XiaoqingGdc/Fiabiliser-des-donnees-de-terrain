# ☀️ Voltéo Énergies - Fiabiliser des données de terrain 

## 🎯 Objectif du Projet
Ce projet a été réalisé dans le cadre du nettoyage chez Voltéo Énergies. Chaque jour, les capteurs de nos parcs solaires et éoliens exportent des données de production brutes. Cet export est initialement "sale" (unités mélangées, virgules décimales, valeurs manquantes ou aberrantes, doublons, etc.). 

L'objectif de ce projet est de concevoir un pipeline automatisé en Python permettant de récupérer cet export brut, d'appliquer une grille de nettoyage stricte à l'aide de la bibliothèque `pandas et numby`, et de livrer un jeu de données parfaitement fiable et standardisé.
